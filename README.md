# 欧式几何游戏截图批量处理脚本

### 需求来源

最近发现点灯机器人游戏蛮好玩的，沉迷其中无法自拔。决定将每一关的解法截图记录下来做个攻略。但是由于我使用的手机是小米8手机(刘海屏)，截图后，图像两边有黑边。因此，编写此脚本，批量的去除截图的黑边，以及批量的对图片进行重命名。
